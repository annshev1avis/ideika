# Проект Идейка
---
## Что это такое: 
API для небольшой соцсети "Идейка" (похоже на Pinterest, но в ней делятся не фотографиями, а идеями, чем заняться)
API присылает информацию о пользователях и карточках (публикациях) в формате JSON

## На чем написано:
Python - Django

## История:
Это учебный проект для изучения нового фреймворка
